How to Use the App

Welcome! Follow these simple steps to get started with our app:

  1. Data Visualization: Click the click here button or the Image to enter and accec the data visualization tool. Upload your file and select the graph your need the viola... you will get your graph and can download it as an image.
  2. Data Collection: Click the click here button or the Image. First, enter the file name. Then, in the second field, enter all the column names, separated by commas (e.g., Name, Age, Email). Click Enter — a form will appear with input fields for each column. You can now store as many records as needed. The data can be downloaded anytime and accessed again by entering the file name. You can also add new data to the file anytime.
  3. Report: Click the click here button or the Image. Upload file for a summary report and you can also download it if needed.
  4. 3-D Graph: It is a 3-D graph generator.
  5. Feedback: Write your Feedback here and ask for any update you need according to your need and check in the Upcomming section for update.
